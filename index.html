Sobre o evento

Java Day será um evento dedicado a desenvolvedores que desejam se aprimorar em Java e tecnologias relacionadas.

Com palestras de renomados profissionais da área, o evento é uma oportunidade única para aprender, trocar experiências e fazer networking com outros desenvolvedores.

Tem como origem o evento do Java Noroeste em 2019.
